# 대학리뷰

대학리뷰는 학교 홍보자료를 다시 쓰는 글이 아니다.  
**대학의 전략, 정책사업, 교육과정 개발, 학과 구조, 지역·산업 연계가 실제로 어떻게 맞물리는지**를 경영컨설팅과 교육공학 관점에서 읽는 글이다.

## 대학리뷰에서 보는 기준

| 기준 | 질문 |
|---|---|
| 대학의 방향 | 이 대학은 어떤 산업·지역·전공 축으로 자신을 설명하는가? |
| 정책사업 | [[RISE]], [[글로컬대학30]], 국책사업이 대학의 실제 전략과 연결되는가? |
| 교육과정 개발 | 학과 이름이 아니라 실제 커리큘럼, 모듈, 마이크로디그리, 실습, 성과평가 구조가 무엇인가? |
| 조직 구조 | 학과·센터·사업단·지역기관이 분절되어 있는가, 실행 체계로 묶이는가? |
| 지역·산업 연결 | 산학협력, 취업처, 지역정주, 지자체·기업 협력이 구체적인가? |
| 리스크 | 홍보 문구와 실제 운영 사이의 간극, 충원·재정·성과관리 리스크는 무엇인가? |

## 대학리뷰를 읽는 법

대학을 볼 때는 “좋은 학교냐 나쁜 학교냐”보다 먼저 봐야 할 것이 있다.

1. 이 대학이 어떤 외부 변화에 대응하고 있는가?
2. 정책사업과 자체 전략이 같은 방향을 보고 있는가?
3. 학과·교육과정·산학협력·취업 지원이 하나의 스토리로 연결되는가?
4. 지역·산업·학생 모집을 동시에 설명할 수 있는 포지션이 있는가?
5. 실행 조직과 성과관리 체계가 전략을 따라가고 있는가?

대학 이름보다 중요한 것은 **대학의 전략이 실제 운영 구조, 교육과정, 학습경험으로 구현되고 있는지**다.
